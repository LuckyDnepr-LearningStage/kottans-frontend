1. Настройки и синтаксис

  • Условия набора правил:
    • Селектор — начало набора правил, используемого для нацеливания на элемент, которому будет присвоен стиль.
    • Блок объявлений — код между (и включая) фигурными скобками ({ }), который содержит объявления CSS.
    • Объявление — имя группы для пары свойств и значений, которая применяет стиль к выбранному элементу.
    • Свойство — первая часть объявления, указывающая, какая визуальная характеристика элемента должна быть изменена.
    • Значение — вторая часть объявления, обозначающая значение свойства.

  • Термины встроенного стиля:
    • Открывающий тег — начало элемента HTML. Это элемент, который будет стилизован.
    • Атрибут — атрибут стиля используется для добавления встроенных стилей CSS к элементу HTML.
    • Объявление — имя группы для пары свойств и значений, которая применяет стиль к выбранному элементу.
    • Свойство — первая часть объявления, указывающая, какая визуальная характеристика элемента должна быть изменена.
    • Значение — вторая часть объявления, обозначающая значение свойства.
  • Встроенные стили CSS могут быть записаны внутри открывающего тега HTML с использованием атрибута стиля.
  • Встроенные стили можно использовать для оформления HTML, но это не лучшая практика.
  • Внутренняя таблица стилей записывается с использованием элемента <style> внутри Элемент <head> HTML-файла.
  • Внутренние таблицы стилей можно использовать для стилизации HTML, но это также не лучшая практика.
  • Внешняя таблица стилей отделяет код CSS от HTML с помощью расширения файла «.css».
  • Внешние таблицы стилей — лучший подход, когда речь идет об использовании HTML и CSS.
  • Внешние таблицы стилей связаны с HTML с помощью элемента <link>.

2. Селекторы

  • CSS может выбирать элементы HTML по типу, классу, идентификатору и атрибуту.
  • Все элементы можно выбрать с помощью универсального селектора (* {}).
  • Элемент может иметь разные состояния с помощью селектора псевдокласса (:before, :after, :hover и т.д.).
  • К одному элементу HTML можно применить несколько классов CSS.
  • Классы можно использовать повторно, а идентификаторы можно использовать только один раз.
  • Идентификаторы имеют более высокий приоритет, чем классы, а классы более более высокий приоритет, чем тип. Это означает, что идентификаторы переопределяют любые стили из класса, а классы переопределяют любые стили из селектора типа.
  • Несколько селекторов могут быть объединены в цепочку для выбора элемента. Это повышает сложность, но может быть необходимо.
  • Вложенные элементы можно выбирать, разделяя селекторы пробелом.
  • Несколько несвязанных селекторов могут получить одинаковые стили, разделив имена селекторов запятыми.

3. Визуальные правила

  • Свойство font-family определяет шрифт элемента.
  • font-size управляет размером отображаемого текста.
  • font-weight определяет, насколько тонким или толстым будет отображаться текст.
  • Свойство text-align размещает текст слева, справа или в центре родительского контейнера.
  • Текст может иметь два различных атрибута цвета: color и background-color. color определяет цвет текста, а background-color определяет цвет позади текста.
  • CSS может сделать элемент прозрачным с помощью свойства opacity.
  • CSS также может установить фон элемента на изображение с помощью свойства background-image.
  • Флаг !important перекрывает любой стиль, однако его почти никогда не следует использовать, так как переопределить.

4. Box model

  • width and height: ширина и высота области содержимого.
  • padding: расстояние между областью содержимого и границей.
  • border: толщина и стиль границы, окружающей область содержимого и отступы.
  • margin: расстояние между границей и внешним краем элемента.
  • Box model включает набор свойств, используемых для создания пространства вокруг элементов HTML и между ними.
  • Height and Width области содержимого можно задать в пикселях или процентах.
  • Borders окружают область содержимого и padding элемента. Цвет, стиль и толщину границы можно задать с помощью свойств CSS.
  • Padding – это пространство между областью содержимого и границей. Его можно задать в пикселях или процентах.
  • Margin – это расстояние за пределами границы элемента.
  • Horizontal margins добавляются, поэтому общее пространство между границами соседних элементов равно сумме правого поля одного элемента и левого поля соседнего элемента.
  • Vertical margins схлопываются, поэтому расстояние между вертикально смежными элементами равно большему полю.
  • margin: 0 автоматически центрирует элемент по горизонтали внутри его родительской области содержимого, если он имеет ширину.
  • Свойство overflow может иметь значение display, hide или scroll и определяет, как HTML будет отображать содержимое, выходит за пределы области содержимого своего родителя.
  • Свойство visibility может hide или show элементы.

5. Changing the Box Model (изменение Box Model)

  • Многие свойства в CSS имеют значение по умолчанию и не должны быть явно установлены в таблице стилей.
  • Например, вес шрифта по умолчанию для текста нормальный, но эта пара свойство-значение обычно не указывается в таблице стилей.
  • То же самое можно сказать и о блочной модели, которую предполагают браузеры. В CSS свойство box-sizing управляет типом блочной модели, которую браузер должен использовать при интерпретации веб-страницы.
  • Значение по умолчанию для этого свойства — content-box. Это та же блочная модель, на которую влияет толщина границ и отступы.
  • К счастью, мы можем сбросить всю модель блока и указать новую: border-box.

    • * {

    box-sizing: border-box;

    }

  • Код в приведенном выше примере сбрасывает блочную модель к границе для всех элементов HTML. Эта новая Box Model позволяет избежать размерные проблемы, которые существуют в прежней блочной модели, о которой вы узнали.
  • В этой Box Model высота и ширина останутся фиксированными. Толщина границы и отступы будут включены внутри Box Model, а значит габаритные размеры коробки не меняются.

    • <h1>Hello World</h1>

    * {

    box-sizing: border-box;

    }


    h1 {

    border: 1px solid black;

    height: 200px;

    width: 300px;

    padding: 10px;

    }

  • В приведенном выше примере высота блока останется равной 200 пикселям, а ширина — 300 пикселям. Толщина границы и отступ останутся полностью внутри блока.

6. Display and positioning (отображение и позиционирование)

  • Положение элемента по умолчанию можно изменить, установив его свойство position. Свойство position может принимать одно из пяти значений:
    • static - the default value (it does not need to be specified)
    • relative
    • absolute
    • fixed
    • sticky
  • Свойство position позволяет указать положение элемента.
  • Если задано значение relative, позиция элемента определяется относительно его позиции по умолчанию на странице.
  • Если задано значение absolute, позиция элемента определяется относительно его ближайшего позиционированного родительского элемента. Его можно прикрепить к любой части веб-страницы, но элемент все равно будет перемещаться вместе с остальной частью документа при прокрутке страницы.
  • Если задано значение fixed, положение элемента можно закрепить в любой части веб-страницы. Элемент останется в поле зрения несмотря ни на что.
  • Если задано значение sticky, элемент может прикрепляться к определенному смещению, когда пользователь прокручивает его родительский контейнер.
  • z-index элемента указывает, насколько далеко назад или вперед элемент отображается на странице, когда он перекрывает другие элементы.
  • Свойство display позволяет вам управлять тем, как элемент перемещается по вертикали и горизонтали в документе.
  • строчные элементы занимают как можно меньше места, и их ширина или высота не могут регулироваться вручную.
  • блочные элементы занимают ширину своего контейнера и могут иметь регулируемую вручную высоту.
  • Элементы встроенного блока могут иметь заданную ширину и высоту, но они также могут располагаться рядом друг с другом и не занимать всю ширину своего контейнера.
  • Свойство float может перемещать элементы максимально влево или вправо на веб-странице.
  • Вы можете очистить левую или правую сторону элемента (или обе), используя свойство clear.

7. Цвета

Существует четыре способа представления цвета в CSS:
    Именованные цвета — существует более 140 именованных цветов, которые вы можете просмотреть здесь. HEX - шестнадцатеричные цвета
    • Шестнадцатеричная система счисления состоит из шестнадцати цифр, от 0 до 9, за которыми следуют буквы от «А» до «F».
    • Шестнадцатеричные значения всегда начинаются с # и определяют значения красного, синего и зеленого с использованием шестнадцатеричных чисел, таких как #23F41A.
    • Шестизначные шестнадцатеричные значения с повторяющимися значениями для каждого значения RGB можно сократить до трех цифр.
    RGB
    • Цвета RGB используют синтаксис rgb() с одним значением для красного, одним значением для синего и одним значением для зеленого.
    • Значения RGB находятся в диапазоне от 0 до 255 и выглядят так: rgb(7, 210, 50).
    HSL
    • HSL означает оттенок (сам цвет), насыщенность (интенсивность цвета) и легкость (насколько светлым или темный цвет).
    • Оттенок варьируется от 0 до 360, а насыщенность и яркость представлены в процентах следующим образом: hsl(200, 20%, 50%).
    • Вы можете добавить непрозрачность к цвету в RGB и HSL, добавив четвертое значение, a, которое представлено как процент.

8. Типографика

  • Типографика — это искусство расположения текста на странице.
  • Текст может быть полужирным или тонким с помощью свойства font-weight.
  • Текст может быть выделен курсивом с помощью свойства font-style.
  • Расстояние между строками текста по вертикали можно изменить с помощью свойства line-height.
  • Шрифты Serif имеют дополнительные детали на концах каждой буквы. Шрифты Sans-Serif — нет.
  • Резервные шрифты используются, когда определенный шрифт не установлен на компьютере пользователя.
  • Свойство word-spacing изменяет расстояние между отдельными словами.
  • Свойство letter-spacing изменяет расстояние между отдельными буквами.
  • Свойство text-align изменяет горизонтальное выравнивание текста.
  • Google Fonts предоставляет бесплатные шрифты, которые можно использовать в файле HTML с тегом <link< или свойством @font-face.
  • Локальные шрифты можно добавить в CSS с помощью свойства @font-face и пути к источнику шрифта.

    @font-face {

    font-family: 'GlegooBanner';

    src: url('../fonts/Glegoo-Regular.woff2') format('woff2'),

    url('../fonts/Glegoo-Regular.woff') format('woff'),

    url('../fonts/Glegoo-Regular.ttf') format('truetype')

    }
